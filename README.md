# FoP-Lab-Assignments
Foundations Of Programming (22 lab Assignments)

# C Programming Lab Assignments

1. Check whether a given year is a leap year.

2. Write a menu driven program to perform basic arithmetic operations.

3. Generate multiplication table of a given number.

4. Calculate salary of an employee using basic pay, HRA, TA, gross salary and net salary.

5. Perform matrix operations:
   - Addition
   - Saddle point
   - Inverse
   - Magic square

6. Compute factorial of a number with and without recursion.

7. Accept student details using array of structures and display their result.

8. Accept marks of five subjects and compute result and grade.

9. Check whether a number is an Armstrong number.

10. Implement a menu driven calculator to perform:
    - Addition
    - Subtraction
    - Multiplication
    - Division
    - Power (x^y)
    - Factorial (x!)

11. Perform number operations:
    - Square root
    - Square
    - Cube
    - Prime check
    - Factorial
    - Prime factors

12. Find smallest divisor and Greatest Common Divisor (GCD) of two numbers.

13. Accept a number and print digits in reverse order.

14. Convert a binary number into decimal number.

15. Generate pseudo random numbers.

16. Accept a list of N integers and partition them into even and odd numbers.

17. Compute sum of sine series.

18. Perform string operations:
    - Length
    - Reverse
    - Equality check
    - Palindrome check
    - Substring check

19. Create structure EMPLOYEE and compute:
    - Total number of employees
    - Count of male and female employees
    - Employees with salary more than 10,000
    - Employees with designation "Asst Manager"

20. Swap two numbers with and without pointers.

21. Copy contents of one file to another using file handling.

22. Write a menu driven program to perform string operations using inbuilt functions.
